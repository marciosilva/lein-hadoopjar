# lein-hadoopjar

A Leiningen plugin to do many wonderful things.

## Usage

Put `[lein-hadoopjar "0.1.0-SNAPSHOT"]` into the `:plugins` vector of your project.clj.

Run lein hadoopjar to build a job jar:

    $ lein hadoopjar
